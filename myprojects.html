Project 1: TextAdventure

In project TextAdventure moest ik een game maken met alleen tekst. Eisen zijn heel belangerijk en daarom heb ik dit project stap voor stap gedaan zodat ik alle eisen erin had gestopt. Mijn idee voor dit project was om door een huis te zoeken naar een eindbaas. In dit huis zijn allemaal vijanden die je moet verslaan door een combat system waarbij je zelf kiest hoe je aanvalt. Ook zijn er een paar mini bosses die lastiger zijn om te verslaan. Sneaky zijn is ook een optie door de vijand niet af te maken op een luide manier bijvoorbeeld met een wapen. Ik moest een cover-art maken die past bij het spel. Ik heb gekozen om te laten zien dat je als een soort John Wick aan het spelen bent. Aan de andere kant staat de eindbaas die een viscacha is. Het was namelijk ook een eis om een viscacha in je spel te doen. Daarom heet mijn spel ook Viscacha Hunt. Hier zie je foto's van het project.
