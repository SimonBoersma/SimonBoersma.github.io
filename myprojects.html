Info Projecten

Op de opleiding Software Developer in het ROC van Twente heb ik veel projecten gemaakt in jaar 1 en begin jaar 2.
Ik heb veel projecten gemaakt met C# en Unity. Hierdoor heb ik veel geleerd over game-development.
Op deze pagina ga ik projecten behandelen die ik heb gemaakt.

Project 1: TextAdventure

In project TextAdventure moest ik individueel een game maken met alleen tekst. Eisen zijn heel belangerijk en daarom heb ik dit project stap voor stap gedaan zodat ik alle eisen erin had gestopt. Mijn idee voor dit project was om door een huis te zoeken naar een eindbaas. In dit huis zijn allemaal vijanden die je moet verslaan door een combat system waarbij je zelf kiest hoe je aanvalt. Ook zijn er een paar mini bosses die lastiger zijn om te verslaan. Sneaky zijn is ook een optie door de vijand niet af te maken op een luide manier bijvoorbeeld met een wapen. Ik moest een cover-art maken die past bij het spel. Ik heb gekozen om te laten zien dat je als een soort John Wick aan het spelen bent. Aan de andere kant staat de eindbaas die een viscacha is. Het was namelijk ook een eis om een viscacha in je spel te doen. Daarom heet mijn spel ook Viscacha Hunt. Hier zie je foto's van het project.

Project 2: Arcade Game

Project Arcade Game was het eerste grote project waar ik in een groep moest werken. We moesten in Unity een game speciaal maken voor een arcade. Eisen waren weer het belangerijkste dus daarom gingen we werken met een programma genaamd Trello. Hier plannen we alle deadlines en alle taken die we hebben verdeeld over elkaar. Elk groepje kreeg ook willekeurig een paar extra eisen. Zo kregen wij
Doelgroep: vrouwen
Limitatie: spel kunnen spelen met één knop
Onderwerp: koken
Ik heb het meeste gewerkt aan dat veschillende knoppen werken op scenes (MainMenu, GameScene, GameOver).
